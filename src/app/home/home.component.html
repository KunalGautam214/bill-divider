How to sign up?

    Follow the below steps to create an account.
  1. Click on power icon.
  2. Click on Sign up.
  3. Enter member names and hit enter button to add member in the group.
  4. Enter group name and password and hit Create button to create a new group.
  5. Credentials share among the group members for further activities.

The idea behind this web site

If you use this web site, you no longer have to use pen and paper for record maintenance. Moreover, you can access it anywhere anytime.
